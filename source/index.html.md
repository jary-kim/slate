# Introduction
WaaS API를 사용하여 WEB 3.0에 쉽게 접근할 수 있습니다.

MPC API 사용하여 복잡한 과정 없이 간편하게 MPC 지갑을 생성하고 관리할 수 있습니다.
MPC 지갑은 키 정보를 안전하게 보관하고 복구할 수 있습니다.
